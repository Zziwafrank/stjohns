# St John’s Senior Secondary School Matugga Website

## 📌 Overview

This is the official school website for **St John’s Senior Secondary School Matugga**.
It is designed to provide information about the school, showcase activities, and improve communication between the school, students, and parents.

## 🎯 Purpose

The website helps to:

* Share school information and updates
* Display student activities and achievements
* Provide admission details
* Allow easy contact with the school

## 🧩 Features

* 🏠 Home page with welcome message
* 🖼️ Gallery section for school events
* 👨‍🏫 About page with school background
* 📚 Services/Academics section
* 📞 Contact page with location and details
* 📱 Fully responsive design (works on phone & computer)

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (if added)

## 🚀 How to Use

1. Download or open the project files
2. Open the `index.html` file in any web browser
3. Navigate through the pages using the menu
